A simple CRUD web application for collecting and managing feedback for BNCC internal events.
Built with Express.js and JSON file for storage.

# Tech Stack
- Frontend: HTML, CSS, Javascript, Bootstap
- Backend: Express.js, Node.js, CORS
- Storage: JSON file (data.json)
 
 
# Installation
1.	Clone the repository  
	git clone https://github.com/mchristabela/rollingweek-rnd  
	cd RnD  
2.	Install backend dependencies  
	cd bncc-feedback-backend  
	npm install  
 
# How to Run the Server
1.	Start the backend server  
	cd bncc-feedback-backend  
	node server.js  
	- You should see:  Server berjalan di http://localhost:3000  
2.	Open the frontend
   - Open bncc-feedback-frontend/index.html in your browser
   - Or use a live server extension
